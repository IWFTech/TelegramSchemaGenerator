Recent changes

June 11, 2026

Bot API 10.1

  • Added the method sendMessage.

Getting updates

Update

This object represents an incoming update. At most one of the optional fields can be present in any given update.

Field Type Description
update_id Integer The update's unique identifier.
message Message Optional. New incoming message of any kind.

getUpdates

Use this method to receive incoming updates using long polling. Returns an Array of Update objects.

Notes 1. This method will not work if an outgoing webhook is set up.

Parameter Type Required Description
offset Integer Optional Identifier of the first update to be returned.

Available types

BotName

This object represents the bot's name.

Field Type Description
name String The bot's name

CallbackGame

A placeholder, currently holds no information.

Message

This object represents a message.

Field Type Description
message_id Integer Unique message identifier.

InlineKeyboardMarkup

This object represents an inline keyboard.

Field Type Description
inline_keyboard Array of String Button rows.

InlineKeyboardButton

This object represents one button of an inline keyboard.

Field Type Description
text String Label text on the button.
style String Optional. Style of the button. Must be one of "danger" (red), "success" (green) or "primary" (blue).

KeyboardButton

This object represents one button of the reply keyboard.

Field Type Description
text String Text of the button.
style String Optional. Style of the button. Must be one of "danger" (red), "success" (green) or "primary" (blue).

Chat

This object represents a chat.

Field Type Description
type String Type of the chat, can be either "private", "group", "supergroup" or "channel".

ChatMember

This object contains information about one member of a chat. Currently, the following 3 types of chat members are supported:

  • ChatMemberOwner
  • ChatMemberAdministrator
  • ChatMemberMember

ChatMemberOwner

Represents a chat member that owns the chat.

Field Type Description
status String The member's status in the chat, always "creator".
user User Information about the user.

ChatMemberAdministrator

Represents a chat member that has administrator privileges.

Field Type Description
status String The member's status in the chat, always "administrator".
user User Information about the user.

ChatMemberMember

Represents a regular chat member.

Field Type Description
status String The member's status in the chat, always "member".
user User Information about the user.

BotCommandScope

This object represents the scope to which bot commands are applied. Currently, the following 3 scopes are supported:

  • BotCommandScopeDefault
  • BotCommandScopeAllPrivateChats
  • BotCommandScopeChatMember

BotCommandScopeDefault

Represents the default scope of bot commands.

Field Type Description
type String Scope type, always "default".

BotCommandScopeAllPrivateChats

Represents the scope of all private chats.

Field Type Description
type String Scope type, always "all_private_chats".

BotCommandScopeChatMember

Represents the scope of commands for a specific member of a specific chat.

Field Type Description
type String Scope type, always "chat_member".

PassportElementError

This object represents an error in the Telegram Passport element which was submitted that should be resolved by the user. It should be one of:

  • PassportElementErrorDataField
  • PassportElementErrorFile

PassportElementErrorDataField

Represents an issue in one of the data fields.

Field Type Description
source String Error source, must be "data".

PassportElementErrorFile

Represents an issue with a document scan file.

Field Type Description
source String Error source, must be "file".

InlineQueryResult

This object represents one result of an inline query. Currently, the following 2 result types are supported:

  • InlineQueryResultPhoto
  • InlineQueryResultCachedPhoto

InlineQueryResultPhoto

Represents a link to a photo.

Field Type Description
type String Type of the result, always "photo".
photo_url String A valid URL of the photo.

InlineQueryResultCachedPhoto

Represents a link to a photo stored on Telegram servers.

Field Type Description
type String Type of the result, always "photo".
photo_file_id String A valid file identifier of the photo.

ReactionType

This object describes the type of a reaction. Currently, it can be one of ReactionTypeEmoji, ReactionTypeCustomEmoji and ReactionTypePaid.

ReactionTypeEmoji

The reaction is based on an emoji.

Field Type Description
type String Type of the reaction, always "emoji".

ReactionTypeCustomEmoji

The reaction is based on a custom emoji.

Field Type Description
type String Type of the reaction, always "custom_emoji".

ReactionTypePaid

The reaction is paid.

Field Type Description
type String Type of the reaction, always "paid".

ReplyKeyboardMarkup

This object represents a custom keyboard with reply options.

Field Type Description
keyboard Array of String Array of button rows.

ReplyKeyboardRemove

Upon receiving a message with this object, Telegram clients will remove the current custom keyboard.

Field Type Description
remove_keyboard Boolean Requests clients to remove the custom keyboard. Must be True.

ForceReply

Upon receiving a message with this object, Telegram clients will display a reply interface to the user.

Field Type Description
force_reply Boolean Shows reply interface to the user. Must be True.

Available methods

getMe

A simple method for testing your bot's authentication token. Requires no parameters. Returns basic information about the bot in form of a User object.

getMyName

Use this method to get the current bot name for the given user language. Returns BotName on success.

Parameter Type Required Description
language_code String Optional A two-letter ISO 639-1 language code or an empty string

getWebhookInfo

Use this method to get current webhook status. Requires no parameters. On success, returns a WebhookInfo object.

sendMessage

Use this method to send text messages. On success, the sent Message is returned.

Parameter Type Required Description
chat_id Integer or String Yes Unique identifier for the target chat.
text String Yes Text of the message to be sent.
reply_markup InlineKeyboardMarkup or ReplyKeyboardMarkup or ReplyKeyboardRemove or ForceReply Optional Additional interface options.

Games

setGameScore

Use this method to set the score of the specified user in a game message. On success, if the message is not an inline message, the Message is returned, otherwise True is returned.

Parameter Type Required Description
user_id Integer Yes User identifier.
score Integer Yes New score.

User

This object represents a Telegram user or bot.

Field Type Description
id Integer Unique identifier for this user or bot.
is_bot Boolean True, if this user is a bot.

WebhookInfo

This object represents webhook info.

Field Type Description
url String Webhook URL.
